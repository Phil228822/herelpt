# Context

## What is this?

## Tech Stack
- Pure HTML/CSS/JS (no framework, no build step)
- Fonts: Schibsted Grotesk (body), Fustat (headlines), Caveat (testimonial names)
- Icons: Lucide CDN (but user hates icon boxes - use sparingly or not at all)
- Video: CloudFront-hosted trail video for homepage hero
- Hosting target: Netlify/Vercel/GitHub Pages (free static hosting)
